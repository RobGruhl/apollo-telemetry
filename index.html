Reading the numbers

  • Mobile visits ≈ how many kids. Scouts scan the QR code with phones, so the headline counts only US mobile sessions that also pass the bot filter. Desktop traffic is reported separately — sophisticated crawlers wear desktop-browser disguises, and there is no advertising that would bring real desktop crowds.
  • The service worker hides repeat visits. After the first load, the whole site is cached on the phone, so a scout replaying the mission offline generates no new traffic. First visits are counted reliably; browsing depth is undercounted.
  • The bot weather report is deliberately separate. Those numbers include crawlers that wear convincing browser disguises — they are context, not headcount. Every headline number on this page is phones-only.
  • This page refreshes itself every hour — a GitHub Actions job pulls fresh Cloudflare analytics and commits the update; the repo's history is the measurement log. The timestamps below tell you how fresh the numbers are.
